# Pre-Filing Checklist

## Identity & caption (every document)
- [ ] Correct court name and address
- [ ] Case number correct and identical on every document
- [ ] Caption parties spelled correctly, roles correct
- [ ] Document title matches what the notice/motion says it is
- [ ] Hearing date/time/department consistent across all documents (if set)

## Content mechanics
- [ ] No placeholder text, brackets, highlights, or comments left in
- [ ] Page limits met (per document type, per local rules)
- [ ] All internal cross-references resolve (sections, exhibits, page cites)
- [ ] Every exhibit referenced is attached, labeled per local convention, in order
- [ ] Exhibit index matches actual exhibits
- [ ] Table of contents / authorities present if required at this length, page numbers accurate
- [ ] Proposed order matches the relief requested in the motion

## Execution
- [ ] Signature block on every document requiring one, correct format
- [ ] Signatures dated; dates make sense (not before events described)
- [ ] Verification / declaration under penalty of perjury in the exact required wording
- [ ] Notarization complete where required: seal, commission expiry, venue line
- [ ] Pro se signer info: name, address, phone, email as local rules require

## Filing & service
- [ ] Filing fee amount confirmed current, or fee waiver order attached
- [ ] Proof of service completed, correct method, correct addresses, signed
- [ ] Service deadline math correct (court vs calendar days, method extensions)
- [ ] Courtesy copy to judge if standing orders require
- [ ] Copies: original + required count for endorsement/return
- [ ] E-filing: PDF text-searchable, bookmarked if required, within size limit
- [ ] Protected identifiers redacted (SSN, financial accounts, minors, DOB)
- [ ] AI-disclosure statement included if the court/judge requires one

## Final
- [ ] Filing deadline recomputed from the rule, not from memory
- [ ] Calendar entry created for expected response deadline / next event
